 # Sprocket Design and Topology Optimization

## Overview
Designed and topology-optimized 18T and 30T power transmission sprockets for a hybrid go-kart drivetrain using SolidWorks.

## Key Contributions
- Torque-based design calculations
- CAD modelling in SolidWorks
- Finite Element Analysis (FEA)
- Topology optimization
- Manufacturing drawing generation
- Material selection (6082-T6 Aluminium)

## Tools Used
- SolidWorks
- SolidWorks Simulation

## Results
- Successfully reduced component mass through topology optimization
- Maintained structural integrity under design loads
- Generated manufacturable final designs
