# Sprocket Design and Topology Optimization

## Overview
Designed and topology-optimized 18T and 30T power transmission sprockets for a hybrid go-kart drivetrain using SolidWorks. The project involved engineering calculations, CAD modelling, FEA validation, and topology optimization to reduce weight while maintaining structural integrity and manufacturability.

## Key Contributions
- Torque-based design calculations
- CAD modelling in SolidWorks
- Finite Element Analysis (FEA)
- Topology optimization
- Manufacturing drawing generation
- Material selection (6082-T6 Aluminium)

## Tools Used
- SolidWorks
- SolidWorks Simulation

## Results
- Achieved **10% weight reduction** for the 18T sprocket through topology optimization.
- Achieved **6% weight reduction** for the 30T sprocket through topology optimization.
- Maintained structural integrity under design loads.
- **Factor of Safety (18T): 6.875**
- **Factor of Safety (30T): 19.92**
- Generated manufacturing-ready CAD models and engineering drawings.
